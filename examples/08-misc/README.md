# Miscellaneous Examples

Other examples that don't fall into the above categories.

## Example List

- **rule_dependency_analysis.rs** - Analyze dependencies between rules
- **rule_file_functions_demo.rs** - Using functions from rule files

## Rule Dependency Analysis

Analyze and visualize dependencies between rules:
- Which rules activate other rules
- Potential circular dependencies
- Rule execution flow
- Optimization opportunities

**Use cases:**
- Debugging complex rulesets
- Performance optimization
- Documentation
- Refactoring

## Rule File Functions

Load and execute functions defined in rule files:
- External function definitions
- Code reusability
- Separation of concerns
- Dynamic function loading

## How to run

```bash
cargo run --example rule_dependency_analysis
cargo run --example rule_file_functions_demo
```
